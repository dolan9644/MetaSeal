# MetaSeal (元印) - 创作者实战与确权指南 🛡️

这是一份写给**每一位原创作者**（画师、写手、程序员、自媒体人）的实战手册。我们不讲深奥的代码，只讲一件事：**如何让剽窃者付出代价，如何让你的心血永远刻着你的名字。**

---

## 📸 模块一：图像确权空间 (Image Protect)

**痛点**：你辛辛苦苦画的图、拍的照片，被别人顺手保存、截图、甚至经过微信的高维压缩后，发到某书某音上说是他自己的原创。如果他用 AI 大模型提取你的画风，你更是百口莫辩。

**MetaSeal 用途**：
这就像是给你的图片打入了一种**人类肉眼绝对看不见、但机器一扫就爆红灯的幽灵荧光剂**。
即使盗图狗把你的图**截图、裁剪、压缩了几十次**，甚至调了滤色，只要他敢发出来，你存下他的图塞进 MetaSeal 的查验器，立刻就能扫出你的名字。

**如何反向维权？**
1. **生成护甲**：首发前，把原图拖进 MetaSeal，一键加密。
2. **提取罪证**：发现有人盗图？把他的图存下来，丢进 MetaSeal 的“版权查验”区。
3. **固定证据**：软件会直接从中提取出“原作者：你的名字、时间戳、唯一哈希”。哪怕只有一点点特征残留，也能揪出这是你的东西，直接拿去向平台发起版权投诉，一击必杀。

**高阶大招（开启 ONNX 对抗性 AI 盾阵）**：
如果你是画师，害怕被拿去炼丹（AI 训练）？打开这个开关。软件会在底层注入针对 AI 视觉的毒药噪声（不影响人类观感）。当爬虫抓取你的图喂给 AI 时，AI 的风格分析矩阵会彻底崩溃，他什么画风也抽不出来。

## 📝 模块二：纯文本/文档防身 (Doc Protect)

**痛点**：你写的干货爆款文章、小说章节、内部不传之秘的文档，只要是文字，最容易被 `Ctrl+C` 加 `Ctrl+V` 一秒偷走，连洗稿都不用。

**MetaSeal 用途**：
我们不搞容易被破解的 PDF 加密。我们使用的是**“零宽隐写术 (Zero-Width Steganography)”**。
简单来说，我们在字与字之间的缝隙里，塞入了宽度为 0 的幽灵字符。小偷复制你的文字、粘贴到任何地方（比如网站、Word、公众号），这些幽灵字符都会像寄生虫一样紧紧跟着被复制过去。

**如何抓洗稿狗？**
1. 把首发文稿 (.txt / .md) 拖入 MetaSeal 保护。你的文章表面上看毫无变化。
2. 小偷全选复制了你的文章，发在知乎。
3. 你把他知乎的文章**全选复制**下来，随便存个文本，拖回 MetaSeal“查验器”。
4. 滴！查验器立刻从表面普通的文字中，提炼出你的隐藏签名。全网文字那么多，只有带有你签名的这段，才是铁证。

## 💻 模块三：源码与机制保护 (Code Protect)

**痛点**：给别人外包写代码，或者开源了部分核心架构，怕别人拔掉你的名字强行占为己有商业化。

**MetaSeal 用途**：
专为程序员和极客打造。拖入任何 `.js, .py, .rs` 等代码文件，系统会基于代码的抽象语法树结构，生成不可逆的正交密码学散列注释块。只要他用了这份核心代码的逻辑，指纹就会永远伴随着项目迭代留存。

## ⛓️ 终极必杀：去中心化归档 (基于区块链与 Arweave)

如果在法庭上或者平台维权时，盗窃者狡辩说：“这图明明是我先画的，里面的水印是你后来用软件伪造加上去的！”怎么办？

这也是为什么我们要引入**“去中心化网络”**的原因。

### 什么是 Arweave 和 OTS？
- **OTS (OpenTimestamps)**：利用比特币网络极高的算力，把你的文件在某年某月某分某秒生成的那一刻，死死地铸造在比特币的区块结构上。这个时间戳**连玉皇大帝也改不了**。这就证明了：“我在这个绝对不可篡改的时间点，就已经拥有了这个文件。”
- **Arweave (永存网络)**：一个去中心化的全球硬盘。只要把文件哈希指纹存进去，只要人类互联网不毁灭，它永远都在，任何人拔插头都没有用。

### 🚨 如何获取密钥，开启上链？ (新手保姆教程)

你不需要买任何加密货币，步骤极其简单：

1. **安装 ArConnect 官方插件钱包**
   - 使用电脑的 Chrome 或 Edge 浏览器，访问 [ArConnect 官网 (arconnect.io)](https://www.arconnect.io/)，点击安装浏览器扩展。
2. **创建一个钱包身份**
   - 打开右上角的小刺猬图标 (ArConnect)，点击 `Create New Wallet`。
   - **重要：** 它会给你展示 12 句英文单词（助记词）。找张纸抄下来，绝对不能丢，也绝对不要给任何人看！
   - 设置一个本地密码，创建完成。
3. **获取并在 MetaSeal 激活**
   - 钱包创建好后，里面可能空空如也（0 AR），但这完全不影响 MetaSeal 把极小的哈希指印上传（前几百 KB 的确权数据在协议层面极大概率是被免费池吸纳的）。
   - （后续系统对接完成后：您只需在 MetaSeal 根目录或者设置面板，导入 ArConnect 导出的 JSON 密钥文件，勾选开启即可！）

一旦开启，从此刻起，您的每一份手稿、每一张分层工程图、每一段代码，都同时接受人类最强密码学网络（比特币网络+去中心化存储）的双重法律级庇护。盗贼面对的不再是您个人，而是全网共识算力筑起的钢铁长城。

# 🎓 MetaSeal 全平台创作者新手操作及上链指南

欢迎来到 MetaSeal 创作者确权节点。
MetaSeal 的核心并非仅仅是一个“图片加水印机”，它是一整套为您打造的**「数字主权捍卫基建」**。所有保护动作都在**您的本地电脑**（物理沙箱）完成，我们不会（也无法）窥探或上传您任何一张未授权的画稿。

但为了让您的作品拥有全球法庭认可的“时间锁”与“永久存档”，您需要了解并进行以下简单的“上链”配置。

---

## 🎯 核心功能一览：你需要开启什么？

### 🛡️ T1 盲水印（无损隐身涂层）
- **适用场景**：所有需要在微信、微博等平台发布的线稿或成图。
- **机制**：开启后，系统会在频域注入您的名字（Author）和版权声明。肉眼绝对看不见，且保证**画质 100% 无损**。不仅如此，即使盗图者将其截图、压缩，MetaSeal 的 `查验版权 (ScannerLab)` 依旧能将其原样提取。

### ☢️ T2 AI 毒化引擎（防爬装甲）
- **适用场景**：您的高清画作不希望被无良数据公司喂给 Midjourney 或 SD。
- **机制**：通过物理层注入隐蔽对抗散列，在人类肉眼无法分辨的像素边缘产生“致盲噪音”。当 AI 爬取并试图学习你的风格时，会导致其模型内部训练出错（梯度爆炸）。
- **提示**：首次使用需在界面点击 **[ 📥 拉取 ONNX 本地离线引擎包 ]**。

## ⛓️ 终极护盾：如何零成本完成“去中心化存证”？

如果您希望在未来发生版权纠纷时，能够拿出一份连反方律师都无法质疑的铁证，您必须开启 **去中心化存证矩阵**。

在开启后，您每次拖图，MetaSeal 都会生成一个叫做 `MetaSeal_Certificate_xxx` 的**法务公证包裹（Bundle）**。要启用其全部威力，请按以下说明配置：

### 第一步：配置您的 Arweave 永存钱包 (JWK)
**Arweave (AR)** 是一条主打“一次付费，永久存储”的区块链。您的原图哈希与包裹会真正刻在人类历史的公证链上。您需要获取一把属于您的钥匙（JWK JSON）。

1. 前往开源 Arweave 钱包：[ArConnect](https://www.arconnect.io/) 或下载其它 AR 浏览器插件。
2. 创建一个新的钱包并妥善保管您的 12位 助记词（Mnemonic）。
3. 进入钱包的 `Settings -> Export Keyfile`（导出密钥文件）。您会下载到一个文本一样的东西（本质上是一串 JSON 代码）。
4. **将该 JSON 中的文本直接复制粘贴**到 MetaSeal 软件 `系统设置 -> 创作者法律署名配置 -> Arweave 底层网关连接 (JWK)` 中。
   > **安全警告**：您的 JWK 只存在于您的本地目录。MetaSeal 是开源的纯本地处理层，绝不会收集秘钥。如果您对其不放心，完全可以拿一个小额的备用空钱包即可！

### 第二步：理解 OpenTimestamps (OTS)
开启 OTS 存证后，软件会将文件的哈希值打包上传至 **比特币 (Bitcoin) 主网**。
- 这个过程**完全免费**！
- 软件会生成带有 `.ots` 后缀的时间戳文件。
- 您在未来拿着 `.ots` 文件进入 `查验版权 (ScannerLab)`，任何人都能独立校验：这份文件确确实实存在于 2026 年比特币的某个区块中。

## 📁 绝对克制的收纳系统：MetaSeal_Vault (极密母盒)

创作者往往有一整个库需要处理，不用担心满屏幕文件夹污染！
MetaSeal 为您设计了极度克制的 **单核归档架构**。

在您指定的出图硬盘下，无论您是一次性框选拖入 100 张画，还是分开 100 天每天传一张，系统都**永远只会创建一个 `MetaSeal_Vault` 母体文件夹**。
在这个母体内，它会为您自动依据原图的名称和精细时间戳设立专属法务子包（如：`[19:00] 柴犬线稿`），里面规规矩矩地码放着该张图的所有确权凭证与区块哈希，随时准备好打包发送给法务团队！

## 🧪 高阶实战：如何向外界“反向证明”您的作品被 AI 窃取？(即梦 Jimeng / SD 盲测协议)

对于开启了 **T2 毒化引擎**的高清大图，最大的意义是对抗未经授权的 AI“炼丹”（风格盗取、垫图重绘）。以下是我们在开发团队内部测试生效的标准实验方案，**强烈建议您亲自操作一次**：

> **实验预期**：证明盗图者在使用你的作品喂给 AI 时，不但学不到你的画风，反而会诱导模型神经元崩溃。

1. **设立双盲对照组**：
   - A 原片：一张线条或构图正常的原画。
   - B 毒片：通过 MetaSeal 开启了 T2 装甲处理后的图（您肉眼看上去与 A 无异）。
2. **极速注入即梦 (Jimeng)**：
   - 【原片对抗】：将 A 图片放入即梦的“以图生图 / 垫图”口，权重拉高，输入 Prompt：“生成一张拥有同样构图元素的赛博朋克画作”。结果大概率会完美融合。
   - 【毒瘤生效】：撤掉 A 图，**换上受保护的 B 毒片**。保持刚才一模一样的垫图权重和 Prompt，点击生成！
3. **见证实战战果 (模型塌陷)**：
   您会发现 AI 解析器遇到了前所未有的“噪音风暴”。因为 T2 毒素针对图像的高频特征波注入了对抗点阵，即梦的解析神经元在扫这幅图时会自动宕机（Gradient Collapse）。它生成的图像极大概率会变成：
   - 极其刺眼的高糊噪点图；
   - 与原动物/物品形态毫无关联的异形马赛克；
   - 色彩断层或崩塌的废片。

### 📌 最后的法务打击 (Reverse-Proofing)
无论盗窃者对您的图进行了截图、压制等多少次传播，只要把在互联网上抓取下来的原片拖进 **[ 查验版权 ScannerLab ]** 槽，只要 T1 印记还在，系统将立马解析出原始作者的名称。带着 OTS 区块链哈希去发起起诉吧！
